Books API Routes

HTTP Method Endpoint Description
GET {{ hostname }}/api/books/ Retrieve a list of all books
POST {{ hostname }}/api/books/ Create a new book entry
GET {{ hostname }}/api/books/{id}/ Retrieve details of a specific book
PUT {{ hostname }}/api/books/{id}/ Update details of a specific book (replace)
PATCH {{ hostname }}/api/books/{id}/ Update details of a specific book (partial)
DELETE {{ hostname }}/api/books/{id}/ Delete a specific book
